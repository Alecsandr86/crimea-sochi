РЕГИСТРАЦИЯ

Для начала работы с сервисом необходимо зарегистрироваться как хостельер. Для этого в поле Регистрации выбираете пункт «отель/пансионат».
Далее ввести регистрационные данные для создания нового пользователя.

*Все поля регистрации обязательны к заполнению.
